Important FAQ

What is the Document Object Model (DOM)?

Dom is a programming interface that represent html documents via structure tree

How do you select an element from DOM?

by writing method like : `getElementById`, `querySelector`, or `getElementsByClassName`.

What is event delegation in the context of the DOM, and why is it useful?

The JavaScript design pattern efficiently manages events on multiple child elements by attaching a single event listener to a common ancestor element.

How do you manipulate an element's and style using the DOM?

the onfocus event handler method is used to change the input element's style when the user clicks inside it.
